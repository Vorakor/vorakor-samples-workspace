Ngx Generic Wizard

Ngx Generic Wizard is the power of a multi-step process in your hands, you define the steps, which pages they go to and watch it do it's magic.

Ngx Generic Wizard currently features a horizontal UI stepper with lines between every step along with a separate component container with the buttons to navigate through the wizard. Configuration is passed in before hand and Ngx Generic Wizard has the unique capability that it can be integrated with stores and APIs. There are several interfaces, the most important ones are ngx-gw-config and ngx-gw-step, as well as ngx-gw-step-status and ngx-gw-step-status-map.

The ngx-gw-config has all of the options necessary for the wizard to function. The steps interface is specifically meant to configure the individual steps with the status and status map interfaces meant to interact with the system using Ngx Generic Wizard.

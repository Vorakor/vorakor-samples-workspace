Ngx Generic Wizard - Inner Workings

Note: I'm not putting all of the models up here just yet, only the main ones a user will actually need to use right away.

Ngx-gw-config Interface

This interface maps out the necessary parameters and configurable options for the Ngx Generic Wizard.

interface INgxGwConfig {
configId?: number;
code?: string[];
description?: string;
baseUrl?: string;
finalizeUrl?: string;
ignoreIncomplete?: boolean;
completedDisabled?: boolean;
applicationId?: number;
moduleId?: number;
}

Ngx-gw-step Interface

This interface maps out the individual steps of the wizard and various configurations for that.

interface INgxGwStep {
stepId?: number;
configId?: number[];
status?: INgxGwStepStatus;
code?: string;
description?: string;
stepOrder?: number;
stepUrl?: string;
ignoreIncomplete?: boolean;
icon?: string;
}

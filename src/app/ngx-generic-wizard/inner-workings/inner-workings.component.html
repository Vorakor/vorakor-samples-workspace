Ngx Generic Wizard - Inner Workings

Note: I'm not putting all of the models up here just yet, only the main ones a user will actually need to use right away. Also note that the widget will not be shown presently as it is not ready for use.

Ngx-gw-config Interface

This interface maps out the necessary parameters and configurable options for the Ngx Generic Wizard.

interface INgxGwConfig {
configId?: number;
code?: string[];
description?: string;
baseUrl?: string;
finalizeUrl?: string;
ignoreIncomplete?: boolean;
completedDisabled?: boolean;
applicationId?: number;
moduleId?: number;
}

Ngx-gw-step Interface

This interface maps out the individual steps of the wizard and various configurations for that.

interface INgxGwStep {
stepId?: number;
configId?: number[];
status?: INgxGwStepStatus;
code?: string;
description?: string;
stepOrder?: number;
stepUrl?: string;
ignoreIncomplete?: boolean;
icon?: string;
}

Ngx-gw-step-status Interface

This interface maps out the status of the steps in the wizard and various states for that.

interface INgxGwStepStatus {
statusId?: number;
code?: string;
description?: string;
}

Ngx-gw-step-status-map Interface

This interface can map whatever status one might have in their database to a status that Ngx Generic Wizard can understand.

interface INgxGwStepStatusMap {
initial?: number;
current?: string;
complete?: string;
incomplete?: string;
}

Ngx-generic-wizard-button-container Component

The button container contains the two buttons that will be used to advance the wizard when a step has been completed or if a user needs to return to a previous step.

Ngx-generic-wizard-step-container Component

The step container is what actually contains the UI for the steps, this component brings together the step component and the line component for a visually appealing experience.

Ngx-gw-action-btn Component

An abstract button that is used by the button container to provide the actions available in navigating through the wizard.

Ngx-gw-step Component

The step is the visual element on the page with a title and step count, one can click on it to provide navigation that way if the configuration is enabled for that.
